Table 表格

用于数据展示的各种样式的表格。

基础用法

表格的展示，注意到需要加.table样式，没有直接对table标签添加样式，因为考虑到某些日期选择器及其他组件会使用table，所以不应该对其有影响。

# 列头1 列头2 列头3 列头4 列头5 列头6
1 行1-1 行1-2 行1-3 行1-4 行1-5 行1-6
2 行2-1 行2-2 行2-3 行2-4 行2-5 行2-6
3 行3-1 行3-2 行3-3 行3-4 行3-5 行3-6

有边框

每个单元格都带有边框。

# 列头1 列头2 列头3 列头4 列头5 列头6
1 行1-1 行1-2 行1-3 行1-4 行1-5 行1-6
2 行2-1 行2-2 行2-3 行2-4 行2-5 行2-6
3 行3-1 行3-2 行3-3 行3-4 行3-5 行3-6

条纹状表格

间隔行带有间隔颜色。

# 列头1 列头2 列头3 列头4 列头5 列头6

3 行3-1 行3-2 行3-3 行3-4 行3-5 行3-6

鼠标悬停

间隔行带有间隔颜色，同时鼠标浮上时做出对应变化。
